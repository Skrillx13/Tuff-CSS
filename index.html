Tuff CSS

This is a fork of the original Tufte CSS project. It adds extra functionality such as support for Syntax Highlighting through PrismJS, using the MathJax library to render Mathematics, adding extra functionaity and fixing styles that are broken (The padding towards the right is broken, and will not work for the original project).
